Interface DataSpecificationIECOptions

An object containing all the parameters for the creation of an instance of the DataSpecificationIEC61360Type ObjectType. All the parameters refers to the standard IEC 61360 and eCl@ss and are not defined in the document "Details of the Asset Administration Shell"
